Topics in Computing a 3 credit hour course appropriate for undergraduate students interested in learning about the essentials of how computers work, and to gain an appreciation for the breadth of computer science as a discipline. Students will gain knowledge in computer hardware, software, advanced features of productivity applications, Web page creation, and algorithmic thinking.

Course Learning Objectives

By the end of the semester, students who successfully complete CSCI 1100-1100L will:

  • Understand how computers work (hardware and software) and understand how the world is changing because of them
  • Have increased knowledge to make informed computing decisions
  • Obtained a high level of proficiency with advanced features of productivity software
  • Be able to build a database and construct SQL queries
  • Be able to build a personal Web site by hand using HTML and CSS
  • Gain exposure to the breadth of the computer science discipline
